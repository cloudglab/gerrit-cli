Gerrit CLI

命令速查

把 Gerrit 变更查看、审查、投票、工作区、CI 构建状态和 AI Skill 工作流接到终端、脚本与发布链路。

GitHub npm

npm install -g @cloudglab/gerrit-cli
初始化 Gerrit 配置
gerrit-cli setup
发布前 smoke
pnpm run release:smoke-query

Wiki / 检索

先做成静态检索：安装、角色入口、审查、CI、工作区、配置、Skill 和发布前 smoke 都能在本页快速过滤。后续可以扩展为构建期索引，覆盖 README、CHANGELOG 和命令注册表。

输入关键词后过滤下方命令。

常用场景

  • 查看我的变更gerrit-cli mine
  • 查看待审查列表gerrit-cli incoming
  • 查看变更详情gerrit-cli show 12345
  • 查看变更 diffgerrit-cli diff 12345
  • 查看评论gerrit-cli comments 12345
  • 搜索变更gerrit-cli search "owner:self status:open"

基础 / 配置

  • 安装 Node.jsnode --version
  • 安装 CLInpm install -g @cloudglab/gerrit-cli
  • CLI 安装入口gerrit install
  • 初始化配置gerrit-cli setup
  • 连接状态gerrit-cli status
  • 更新 CLIgerrit update
  • 卸载预览gerrit uninstall
  • 查看配置gerrit-cli config show
  • 查看版本gerrit-cli version
  • 生成补全gerrit-cli completion bash

角色入口

  • 完整能力gerrit-cli --help
  • 开发者入口gerrit-dev mine
  • 审查者入口gerrit-reviewer incoming
  • CI 入口gerrit-ci build-status 12345 --json
  • 团队 Lead 入口gerrit-lead groups

审查链路

  • 待审查列表gerrit-cli incoming
  • 查看文件列表gerrit-cli files 12345
  • 查看审查人gerrit-cli reviewers 12345
  • 添加评论gerrit-cli comment 12345 -m "LGTM, one nit"
  • 批量行评论echo '[{"file":"src/app.ts","line":42,"message":"Use const"}]' | gerrit-cli comment 12345 --batch
  • 投票通过gerrit-cli vote 12345 --code-review 2 -m "Approved"
  • AI 辅助审查gerrit-cli review 12345 --strategy claude

变更管理

  • 添加审查人gerrit-cli add-reviewer alice@example.com -c 12345
  • 移除审查人gerrit-cli remove-reviewer 12345 alice@example.com
  • 设置 topicgerrit-cli topic 12345 release-0.0.x
  • 重基变更gerrit-cli rebase 12345
  • 合入变更gerrit-cli submit 12345
  • 丢弃变更gerrit-cli abandon 12345
  • 恢复变更gerrit-cli restore 12345

工作区

  • 检出变更gerrit-cli checkout 12345
  • 创建工作区gerrit-cli workspace create 12345
  • 列出工作树gerrit-cli trees
  • 创建变更工作树gerrit-cli tree setup 12345
  • 重基工作树gerrit-cli tree rebase 12345
  • 清理工作树gerrit-cli tree cleanup --dry-run
  • 推送 WIPgerrit-cli push -b main -t refactor --wip

CI / 构建

  • 构建状态gerrit-cli build-status 12345
  • JSON 构建状态gerrit-cli build-status 12345 --json
  • 轮询到完成gerrit-cli build-status 12345 --watch --exit-status
  • 失败链接gerrit-cli failures 12345
  • 提取 Jenkins 链接gerrit-cli extract-url "jenkins"
  • 重新触发构建gerrit-cli retrigger 12345
  • 分析失败gerrit-cli analyze 12345

输出格式

  • 默认文本gerrit-cli show 12345
  • 脚本 JSONgerrit-cli show 12345 --json
  • LLM XMLgerrit-cli show 12345 --xml
  • 搜索 JSONgerrit-cli search "status:merged age:7d" --json
  • 评论 XMLgerrit-cli comments 12345 --xml

组 / 团队

  • 组列表gerrit-cli groups
  • 组详情gerrit-cli groups-show administrators
  • 组成员gerrit-cli groups-members project-reviewers
  • 团队视图gerrit-cli team
  • 按组添加审查gerrit-cli add-reviewer backend-reviewers --group -c 12345

辅助命令

  • 打开变更gerrit-cli open 12345
  • 清理分支gerrit-cli clean --dry-run
  • Cherry-pickgerrit-cli cherry 12345
  • 安装 hookgerrit-cli install-hook
  • 初始化项目gerrit-cli init

环境变量

  • 服务地址export GERRIT_HOST="https://gerrit.example.com"
  • 账号export GERRIT_USERNAME="your-username"
  • HTTP 密码export GERRIT_PASSWORD="your-http-password"
  • smoke 样本变更export GERRIT_SMOKE_CHANGE_ID="12345"
  • smoke 搜索样本export GERRIT_SMOKE_QUERY="status:open"
  • smoke 构建关键词export GERRIT_SMOKE_BUILD_KEYWORD="jenkins"

发布 / Skill

  • 发布前 dry-runpnpm run release:smoke-query
  • 真实只读 smokepnpm run release:smoke-query:live
  • Claude 插件安装/plugin marketplace add cloudglab/gerrit-cli
  • 安装 workflow skill/plugin install gerrit-workflow@gerrit-cli
  • 查看 npm 包npm view @cloudglab/gerrit-cli version
写操作需要显式命令触发；脚本和 AI 集成优先使用 --json 或 --xml 输出。
